simple-ajax-search

-A django based ajax search, without reloading the template.
-Automatically searches the database as you type the search word.

Tools used:
*Django 1.4.3
*Jquery and Ajax
*Twitter bootstrap

A live demo can be found here:
http://simpleajaxsearch.herokuapp.com/
